## 수정 #1 — Hard Boundary Cost 배리어 복구

**파일:** `src/mppi_core.cu:171`  
**심각도:** 🔴 Critical

### 문제
`fmaxf(diff, 1.0e-5f)`로 변경되면서 차량이 경계(collision_radius) 내부로 침범했을 때 hard cost가 침투 깊이에 무관한 상수값(`q_collision * log(2) ≈ 277`)으로 고정되었다.
기존 `fminf`는 `diff`가 음수(실제 침범)일수록 `exp(-30 * diff)`가 지수적으로 증가해 강한 배리어 역할을 했으나, `fmaxf`로 인해 2cm 침범과 10cm 침범이 동일한 비용을 가져 최적화 방향성이 완전히 소멸했다.

### 수정
```cpp
// 수정 전
float capped = fmaxf(diff, 1.0e-5f);

// 수정 후
float capped = fminf(diff, 1.0e-5f);
```

## 수정 #3 — NaN 정제 순서 오류 수정

**파일:** `src/mppi_core.cu:519`  
**심각도:** 🔴 Critical

### 문제
NaN 정제 루프가 `std::min_element` 호출 **이후**에 위치했다. IEEE 754 규칙에 의해 `h_costs_[0]`이 NaN이면 `min_element`는 NaN을 최솟값으로 유지한다. 이때 `min_cost = NaN`이 되고 `std::isinf(NaN) == false`, `NaN >= 1.0e8f == false`이므로 early return 가드를 통과한다. 이후 모든 가중치 계산이 `exp(NaN) = NaN`이 되어 드라이브 토픽에 NaN 명령이 발행된다.

### 수정
NaN 정제 루프를 `min_element` 호출 앞으로 이동.

```cpp
// 수정 전
auto min_it = std::min_element(h_costs_.begin(), h_costs_.end());
float min_cost = *min_it;
// ...
for (int k = 0; k < K_; ++k) {
    if (std::isnan(h_costs_[k])) h_costs_[k] = 1.0e8f;
}

// 수정 후
for (int k = 0; k < K_; ++k) {
    if (std::isnan(h_costs_[k])) h_costs_[k] = 1.0e8f;
}
auto min_it = std::min_element(h_costs_.begin(), h_costs_.end());
float min_cost = *min_it;
```

## 수정 #10 — HOST/CUDA 바운더리 비용 함수 일치

**파일:** `src/smppi_node.cpp:143`  
**심각도:** 🟡 Medium

### 문제
`append_best_traj_costs`의 hard cost 공식이 CUDA `compute_cost_cuda`와 세 가지 불일치 존재:
- 임계값: `collision_radius * 1.2` (CUDA: `1.5`)
- 지수 계수: `-40` (CUDA: `-30`)
- fmaxf/fminf 캡 없음 → 침범 시 `std::exp`가 폭발적으로 증가
- soft cost 계수: `150` (CUDA: `70`)

결과적으로 `/mppi_optimal_trajectory` 토픽의 `boundary_cost` 값이 비현실적으로 커져 텔레메트리 기반 파라미터 튜닝이 불가능했다.

### 수정
HOST 공식을 CUDA와 완전히 일치시킴.

```cpp
// 수정 전
if (min_bnd < mppi_params_.collision_radius * 1.2f)
    hrd = mppi_params_.q_collision *
          std::log(1.f + std::exp(-40.f*(min_bnd - mppi_params_.collision_radius)));
bnd_cost = 150.f * pen * pen + hrd;

// 수정 후
if (min_bnd < mppi_params_.collision_radius * 1.5f) {
    float diff   = min_bnd - mppi_params_.collision_radius;
    float capped = std::min(diff, 1.0e-5f);   // CUDA fminf와 동일
    hrd = mppi_params_.q_collision *
          std::log(1.f + std::exp(-30.f * capped));
}
bnd_cost = 70.f * pen * pen + hrd;
```
